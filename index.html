𐎃 Як перекладати насправді (по кроках) 𐎃
  1. Спочатку розпакуйте ваш мод в якусь папку
  2. Переходьте по шляху resource_pack/texts/en_US.lang (перші два слова "en_US" можуть відрізнятися) ось список мов: en_US - англійська, uk_UA - українська, ru_RU - російська, de_DE - німецька...
  3. Тепер відкривайте файл в якомусь текстовому редакторі (бажано в редакторі для коду, такому як "Acode"), і перекладаєте всі рядки ВАЖЛИВО: ключі "id:item.name" та інші НЕ ПЕРЕКЛАДАТИ, перекладаємо тільки текст після "=". Далі після перекладу зберігаємо файл правильно. Якщо ви перекладали на українську мову - зберігаємо файл як uk_UA.lang
  4. Ще не забудьте перевірити файли предметів у behaviour_pack/items/файл_предмету.json, шукайте там блок "minecraft:display_name": {, там внизу має бути "value" : і якщо там пише ключ з lang файлу - предмет перекладено, а як ні - треба вставляти самому
  5. Будь ласка, на останок перевірте наявність файлу languages.json в resource_pack/texts, а якщо його немає - треба просто створити! Приклад: 
    {
  "uk_UA": "Українська (Україна)",
  "en_US": "English (United States)",
  "en_UK": "English (United Kingdom)",
  "ru_RU": "Русский (Россия)",
  "pl_PL": "Polski (Polska)"
}
